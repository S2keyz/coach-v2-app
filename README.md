# Coach Militaire V2 🎯

## Déployer en 2 minutes (gratuit)

### Option 1 — Vercel CLI (le plus rapide)

1. Installe Node.js si pas déjà fait : https://nodejs.org (bouton LTS)
2. Ouvre un terminal dans ce dossier
3. Tape :

```bash
npm install
npx vercel
```

4. Suis les instructions (compte gratuit, accepte tout par défaut)
5. Tu reçois une URL type `coach-militaire.vercel.app`

### Option 2 — Via GitHub

1. Crée un repo GitHub et push ce dossier
2. Va sur https://vercel.com → "Add New Project"
3. Importe le repo, Framework = Vite
4. Clique Deploy

### Ajouter sur l'écran d'accueil (iPhone)

1. Ouvre l'URL dans **Safari**
2. Bouton Partager (carré + flèche)
3. "Sur l'écran d'accueil"
4. Ajouter

### Ajouter sur l'écran d'accueil (Android)

1. Ouvre l'URL dans **Chrome**
2. Menu ⋮ → "Ajouter à l'écran d'accueil"
